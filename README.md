# 경평 실적보고서 아이템 제목&초안 생성기

AI를 활용하여 공공기관 경영평가 실적보고서의 아이템 제목과 초안 생성을 돕는 웹 애플리케이션입니다. 사용자가 핵심 성과 데이터를 입력하면, 다양한 유형의 제목 제안과 함께 논리적인 구조를 갖춘 보고서 초안을 생성하여 업무 효율을 높여줍니다.

## 주요 기능

-   **AI 기반 제목 및 초안 생성**: Gemini API를 활용하여 창의적이고 전략적인 아이템 제목과 논리적인 보고서 초안을 생성합니다.
-   **다양한 유형 제공**: 명사형, 동사형, 슬로건형 등 15가지 유형의 제목을 제공하여 활용도를 높입니다.
-   **문서 첨부 기능**: 우수 보고서 샘플(.pdf, .docx, .txt 등)을 첨부하여 AI가 내용을 참고하고 비슷한 스타일로 초안을 작성하게 할 수 있습니다.
-   **사용자 친화적 인터페이스**: 단계별 입력 폼을 통해 누구나 쉽게 사용할 수 있습니다.

## 기술 스택

-   **Frontend**: React, TypeScript, Tailwind CSS
-   **Build Tool**: Vite
-   **AI**: Google Gemini API

## 배포

이 프로젝트는 [Vercel](https://vercel.com/)을 통해 배포할 수 있습니다.
